Interactive Image Gallery Website
📌 Project Overview

Interactive Image Gallery is a modern, responsive web application built using HTML, CSS, and JavaScript. The website allows users to browse beautiful image collections, filter images by category, preview images in a lightbox, switch between dark and light themes, and enjoy a smooth user experience across all devices.

🚀 Features
✅ Responsive Design
Works on Desktop, Tablet, and Mobile devices.
Built using CSS Grid and Media Queries.
✅ Image Categories

✅ Interactive Lightbox
Full-screen image preview.

✅ Dark / Light Mode
Theme toggle button.
